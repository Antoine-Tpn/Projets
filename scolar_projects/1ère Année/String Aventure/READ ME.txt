Bonjour, voici le projet IPI nommé String_Aventure confectionné par Antoine Turpin, projet délivré le 29/05/2023.

Pour lancer le jeu vous devez lancer le fichier main.py.

Utilisez les touches q,z,d et p pour vous déplacer et tirer.
Pour quitter le jeu appuyer sur echap.

L'objectif de ce jeu est de trouver et d'atteindre le portail placé dans le niveau:
celle-ci est représenté comme ceci :

                                (-°-)
                                [   ]
                                [   ]
                                [___]

Des ennemis sont placés sur votre chemin, alors éliminer-les !!!
